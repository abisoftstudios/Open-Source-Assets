## 🌟 About This Repo

This is the **official free assets repository** of **Abisoft Studios** — an indie game development studio focused on building games, tools, and resources for the community.

We believe great assets should be **accessible to everyone**, especially indie developers and beginners who are just getting started. Every asset in this repo is:

- ✅ **100% Free** — no hidden costs or premium tiers
- ✅ **Open Source** — MIT licensed, do what you want
- ✅ **Community Driven** — submissions welcome from anyone
- ✅ **Roblox Friendly** — safe for use on Roblox platform
- ✅ **Cross-Platform** — usable in Unity, Unreal, Godot, and more

> 💬 **Join our Discord** to get notified of new asset drops, request assets, and submit your own!

## 🚀 Releases — Asset Drops

We package assets into **themed release drops** so you can download exactly what you need without cloning the whole repo.

### 🏷️ Release Naming Convention

| Tag Format | Example | Meaning |
|---|---|---|
| `roblox-vX.X` | `roblox-v1.0` | Roblox asset pack |
| `gfx-vX.X` | `gfx-v2.1` | GFX / UI pack |
| `scripts-vX.X` | `scripts-v1.3` | Script bundle |
| `audio-vX.X` | `audio-v1.0` | Audio pack |
| `megapack-MONTH-YEAR` | `megapack-april-2025` | Monthly mega drop |
| `unity-vX.X` | `unity-v1.0` | Unity asset pack |

### 📦 Latest Releases

> All releases are listed on the [**Releases Page →**](../../releases)

Each release includes:
- 📄 Full changelog and asset list
- 📁 Zipped asset folders for easy download

## 🗿 Roblox Models

Free 3D models ready to import into Roblox Studio.

```
roblox/models/
├── props/          ← Small decorative objects
├── structures/     ← Buildings, walls, platforms
├── vehicles/       ← Cars, bikes, aircraft
├── characters/     ← NPC rigs, character models
└── nature/         ← Trees, rocks, terrain pieces
```

**How to use:**
1. Download the `.rbxm` file from [Releases](../../releases)
2. Open Roblox Studio
3. Right-click in Explorer → **Insert from File**
4. Select the downloaded `.rbxm`

## 📥 How to Download

### Option A — Download a Release Pack *(Recommended)*
1. Go to [**Releases**](../../releases)
2. Find the pack you want
3. Download the `.zip` attached to that release
4. Extract and use!

### Option B — Clone the Full Repo
```bash
git clone https://github.com/AbishekStudios/free-assets.git
```

### Option C — Download Individual Files
1. Navigate to the file you want
2. Click it to open
3. Click **"Download raw file"** (top right)

## 🤝 Contributing

We welcome community asset submissions! Here's how:

### Via Discord *(Easiest)*
Join our [Discord server](https://discord.gg/VByUYKDWj9) and use the **📦 Asset Hub** panel → **Submit Assets** button.

### Via GitHub Pull Request
1. Fork this repository
2. Add your asset to the correct folder
3. Include a brief `README.md` inside your asset's folder describing what it is
4. Open a Pull Request with the title format: `[CATEGORY] Asset Name`

### Submission Guidelines
- ✅ Must be your original work or properly licensed
- ✅ Must be free for others to use (MIT or CC0)
- ✅ Include a preview image if possible
- ❌ No AI-generated assets without disclosure
- ❌ No stolen or ripped assets
- ❌ No NSFW content
